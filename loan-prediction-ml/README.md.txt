# 💰 Loan Prediction using Machine Learning

## 📌 Project Overview
This project predicts whether a **loan application will be approved** based on applicant details such as income, education, credit history, and loan amount.  
The aim is to build a reliable ML model that helps financial institutions in the loan approval process.

## 🛠️ Tools & Libraries
- Python
- Pandas, NumPy
- Scikit-learn
- XGBoost
- Matplotlib, Seaborn
- Joblib
